#Commentaires

Pour ajouter les commentaires sur un node ou un media : 1. Créer un type de commentaire ou utiliser le commentaire par défaut : /admin/structure/comment 2. Sur le node ou le media, ajouter un champ de type commentaire et sélectionner le type de commentaire.

C'est tout ! (Après pour skinner, il suffit d'étendre les twig des commentaires).
